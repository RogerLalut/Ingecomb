En los archivos html, se agrego el SEO con las siguientes caracteristicas.
	*Descripción: Mediante la etiqueta meta name description, agregué el párrafo explicando 
	en pocas palabras el contenido de mi sitio.

	Palabras Clave: Mediante la etiqueta meta name Keywords, agregué las palabras claves para la busqueda
	Que acompañan a todas las páginas de mi sitio web: Soldadura, construcción, agua, potable, alcantarillado.
